Sublime Text 3

Sublime Text 3 官网是http://www.sublimetext.com/。

配置

1. 配置为中文。

打开软件，按Ctrl + Shift + P，输入Install Package

输入localiz选择ChineseLocalizations

点击后自动安装，安装后跳出中文说明。

即修改成了中文

2.注册or破解


Sublime Text 3 可以免费无限期试用，未注册不影响使用。
注册可购买注册码。破解
修改C:\Windows\System32\drivers\etc\hosts hosts 文件

注册码

3.配置模板。

打开软件，按Ctrl + Shift + P，输入Install Package

打开插件搜索栏

  • SublimeTmpl 插件，快速生成文件模板。

Ctrl+Alt+h 新建 html 文件
Ctrl+Alt+j 新建 javascript 文件
Ctrl+Alt+c 新建 css 文件
Ctrl+Alt+p 新建 php 文件
Ctrl+Alt+r 新建 ruby 文件
Ctrl+Alt+Shift+p 新建 python 文件

在Preferences->Package Settings->SublimeTmpl->Settings User进行模板配置，如：

在SublimeTmpl\templates文件夹中修改模板。

4.安装插件

  • AutoFileName
    自动补全文件名插件
  • Anaconda
    在Preferences->Package Settings->Anaconda->Settings Usert进行模板配置。

当前python.exe路径

输出C:\\Users\\Fuwenyue\\Anaconda3\\python.exe。

配置修改为C:/Users/Fuwenyue/Anaconda3/python.exe

  • OmniMarkupPreviewer
    在Preferences→Package Settings→OmniMarkupPreviewer→Setting——User进行配置。 按输入快捷键：Ctrl+Alt+O，打开浏览器界面预览即可。

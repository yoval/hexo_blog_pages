〇、常用库

from selenium import webdriver 导入webdriver模块

from selenium.webdriver import ActionChains 导入动作链类，动作链可以储存鼠标的动作，并一起执行

from selenium.webdriver.common.key import Key 键盘操作使用的是Keys类,一般配合send_keys使用

from selenium.webdriver.support.select import Select 下拉框的操作都交由Select类进行

from selenium.webdriver.common.by import By by定位

一、基本操作

1. 浏览器相关操作

browser = webdriver.Chrome() 创建浏览器对象

browser.get('https://www.baidu.com/') 打开百度

browser.get_window_size() 获得浏览器尺寸，返回：

browser.maximize_window() 窗口最大化

browser.set_window_size('1050','708') 设置窗口尺寸

browser.get_window_position() 获取浏览器位置

browser.close() 关闭当前窗口

browser.quit() 关闭所有出口

2. 页面相关操作

browser = webdriver.Chrome() 创建浏览器对象

browser.get('https://www.baidu.com/') 打开百度

browser.refresh() 刷新页面

browser.back() 后退

browser.forward() 前进

browser.current_url 当前url

browser.title 标题

browser.get_screenshot_as_file('1.png') 截屏

browser.page_source 网站源码，可直接用BeautifulSoup 解析

3. 元素的操作

browser = webdriver.Chrome() 创建浏览器对象

browser.get('https://www.baidu.com/') 打开百度

browser.find_element_by_id('kw').send_keys('笔札') 输入框输入数据

browser.find_element_by_id('kw').clear() 清空输入框

browser.find_element_by_id('kw').send_keys(Keys.ENTER) 确认，开始搜索

browser.find_element_by_link_text('贴吧').click() 点击贴吧链接

browser.find_element_by_link_text('贴吧').text

OneManager进阶配置（二）

发表于2021-04-17|更新于2021-04-17
|字数总计:579|阅读时长:2分钟|阅读量:

登录后，在管理>设置里有一些自定义设置。

一、后台配置

  • adminloginpage

定义登录地址，设置后就会隐藏登录按钮。

  • background

自定义背景图片，填入一个图片的url地址。

  • customCss

自定义css，作用于网盘所有页面。如


  • customTheme




  • passfile

设置密码文件名，文件内容为设置的密码

  • sitename

网站名称

  • theme

切换主题

  • timezone

设置时区


二、多盘设置

  • diskname

该网盘名称

  • domain_path

设置不同域名打开时访问不同目录

  • downloadencrypt

即便无密码，也可以直接下载文件

  • guestup_path

游客上传路径，可做图床

  • public_path

三、进阶设置

1. 设置网站ico图标

  • 将favicon.ico图片放在网盘1根目录下
  • 在customCss或customScript中插入。

2. 刷新自动切换背景图片

https://api.ixiaowai.cn/api/api.php (二次元动漫)
http://www.dmoe.cc/random.php（二次元随机图）
https://api.ixiaowai.cn/mcapi/mcapi.php （menhera酱）
https://api.ixiaowai.cn/gqapi/gqapi.php （风景）
https://acg.yanwz.cn/wallpaper/api.php（二次元随机图）

3 .利用index.html设置自定义页面

如果一个目录下有名为index.html的文件，则直接显示该文件

4. 设置顶部和底部说明文字

head.md、readme.md

5. head.omf

支持html css js，如设置一言


6. foot.omf

支持html css js，如设置Valine评论

ps，jsdelivr的cdn比unpkg快

<script src='//cdn.jsdelivr.net/npm/valine/dist/Valine.min.js'></script>

7. 设置动态背景

<script size="90" alpha="0.5" zIndex="0" src="https://pan.2bboy.com/ppx/test/ribbon.js" type="text/javascript" charset="utf-8"></script>

白嫖的 EUserv IPV6 服务器使用（已失效）

发表于2021-04-15|更新于2021-04-15
|字数总计:1.2k|阅读时长:6分钟|阅读量:

EUserv 有免费申请的VPS，配置很低，还是IPV6，不经折腾，但还是有些用的。比如用起宝塔面板。

一、连接VPS

由于EUser是IPV6，电脑不支持ipv6话，可以使用heyterm。


二、VPS准备

  1. 修改DNS：echo -e "nameserver 2001:67c:2b0::4\nnameserver 2001:67c:2b0::6" > /etc/resolv.conf，

  2. 安装bt面板：wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh，

  3. 将面板端口设为8080（cloudfare cdn 仅支持个别端口），

  4. 在cloudfare将任意二级域名AAAA解析到vps ip，

  5. 打开宝塔面板，网址类似abc.cde.com:8080，登录宝塔，

  6. 安装python3、pip3：apt install python3 python3-pip，

  7. 安装python的模块：pip3 install requests beautifulsoup4等，

  8. 调整时区：在宝塔面板，Lunix 工具箱将时区设为上海。


三、用vps自己给自己免费续费（延期）

克隆续费代码：git clone https://BdV****mA:f2e821f1d****48b4d371ba01218fe@e.coding.net/fuwenyue/AutoScript/EUserv_extend.git

定时运行：计划任务每月运行一次python3 /root/EUserv_extend/EUserv_extend.py

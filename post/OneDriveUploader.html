用OneDriveUploader上传VPS文件到OneDrive

发表于2021-11-26|更新于2021-11-26
|字数总计:715|阅读时长:3分钟|阅读量:

上一篇用Aria2进行离线下载，下载到VPS硬盘。容量、取回都有些问题，这篇试图将它传到OneDrive方便进行其它操作。此篇参照OneDriveUploader：基于GoLang极为好用的一个OneDrive上传工具。

安装

获取授权链接：

国际版, 个人版(家庭版)访问：https://login.microsoftonline.com/common/oauth2/v2.0/authorize?client_id=78d4dc35-7e46-42c6-9023-2d39314433a5&response_type=code&redirect_uri=http://localhost/onedrive-login&response_mode=query&scope=offline_access%20User.Read%20Files.ReadWrite.All

中国版(世纪互联)访问：https://login.chinacloudapi.cn/common/oauth2/v2.0/authorize?client_id=dfe36e60-6133-48cf-869f-4d15b8354769&response_type=code&redirect_uri=http://localhost/OneDrive-login&response_mode=query&scope=offline_access%20User.Read%20Files.ReadWrite.All

我用的是OneDrive国际版个人版（15G），授权之后：

复制浏览器网址：http://localhost/onedrive-login?code=M.R2_BAY.41e8a024-****-d626-****-d2******27ae


安装OneDriveUploader ：

下载

wget https://raw.githubusercontent.com/MoeClub/OneList/master/OneDriveUploader/amd64/linux/OneDriveUploader -P /usr/local/bin/

更改权限

chmod +x /usr/local/bin/OneDriveUploader

配置

OneDriveUploader -ms -a "http://localhost/onedrive-login?code=M.R2_BAY.41e8a024-****-d626-****-d2******27ae"

显示Init config file: /root/auth.json.表示配置成功。

上传：

上传VPS/DownLoad/Spyder_64bit_full.exe文件到OneDrive的share文件夹。

OneDriveUploader -s "/DownLoad/Spyder_64bit_full.exe" -r "share"

自动运行脚本

在root创建rcloneupload.sh填入代码：（下载目录为上篇的/DownLoad）

更改权限：

chmod +x rcloneupload.sh

运行测试：apt install dos2unix -y，如果报错dos2unix /root/rcloneupload.sh。

修改Aria2配置文件：

*/root/.aria2/aria2.conf *
